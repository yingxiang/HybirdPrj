====================SDK文件说明====================

SDK压缩包中解压后主要有4个部分
       1. sample目录, sample目录主要是包含了SDK的使用demo:sdkDemo和qzoneDemo。应用开发者可以参考sdkDemo来使用IOS SDK 1.8.1。qzoneDemo主要用于QZone开放能力的开发，第三方开发者如果不做这部分能力的开发，不用关注这个demo。
       2. TencentOpenApi_IOS_Bundle.bundle文件，主要是打包了SDK的资源文件。需要添加的应用开发者的工程中。具体添加方法可以参考SDK 1.8.1说明使用文档。
       3. TencentOpenAPI.framework目录, 是SDK 1.8.0的实现文件。需要应用开发者将目录添加到开发工程中。具体添加方法可以参考SDK 1.8.1说明使用文档。
          在TencentOpenAPI.framework目录下Headers目录中，有三个头文件。
          1) sdkdef.h 定义了SDK的字符串常量，错误码定义和APIResponse的定义。
          2) TencentOAuth.h 定义了IOS SDK 1.8.1中封装调用的API定义和API的回调协议定义。
          3) TencentOAuthObject.h 定义了IOS SDK 1.8.1的API调用辅助工具类。主要是帮助应用开发者使用SDK进行应用的开发。 
       4. doc目录, 主要IOS SDK 1.8.1 API文档。使用方法:打开doc/index.html文档即可。
       5. IOS_SDK使用说明(WIN).url, Windows平台打开wiki网站的快捷方式。
       6. IOS_SDK使用说明(MAC).webloc  Mac平台打开wiki网站的快捷方式。
注：引入SDK后，在调用全部接口的情况下会对ipa体积产生300KB左右的增量。
